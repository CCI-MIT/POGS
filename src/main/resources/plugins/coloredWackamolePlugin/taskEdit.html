Configuration
The number of colors ("RED","GREEN","BLUE","BROWN"). They will be used in this order.
The color frequency, should be set separated by commas and must have the values for all colors. Ex: 3 colors : 30,20,50
The color weights used in scoring, should be set separated by commas and must have the values for all colors. Ex: 3 colors : 1,2,3
The total round time in milliseconds.
Mole duration in milliseconds defaults to 1000.
Enter the grid size in total cells ex.: 10x10 = 100 .
If true a random time multiplier will be applied to the mole duration time
